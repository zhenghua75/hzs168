-- SUMMARY --

Integrates jQuery Liquid Carousel features into views slideshow version 7.x.

-- REQUIREMENTS --

* Libraries API 7.x-1.x
* Views Slideshow 7.x-3.x
* jQuery Liquid Carousel 1.0

-- INSTALLATION --

* Install as usual, see (http://drupal.org/documentation/install/modules-themes/modules-7)
  for further information.
* Download and unzip the jQuery Liquid Carousel plugin into 'sites/all/libraries'
  from (http://www.nikolakis.net/liquidcarousel/).
